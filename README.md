## Installation

### 1. Clone the Repository


```bash
git clone https://github.com/your-repo/shotr-server.git
cd shotr-server
```

### 2. Create a `.env` file

```bash
cp .env.example .env
```


### 3. Install Dependencies

```bash
yarn install
```

## Running in Production

```bash
sudo docker compose up -d
```
